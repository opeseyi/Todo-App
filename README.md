# TODO APP

This a Todo List App
Built with **React**

## To use

1. Clone
   ```
   git clone https://github.com/opeseyi/Todo-App
   ```
2. Install dependencies

   ```
   npm install
   ```

3. Run frontend
   ```
   npm start
   ```

### This should be up and runing in your browser and you should be able to interact with it
